# Setup
## Requirements 
- Python 3.13
## Installation
git clone git@github.com:esharpballinger/Distributed-algorithm-testing-tool.git

cd Distributed-algorithm-testing-tool

python3.13 -m venv .venv

source .venv/bin/activate        # Mac/Linux

.venv\Scripts\activate           # Windows

pip install pytest




# Testing
## Run Tests
- pytest
